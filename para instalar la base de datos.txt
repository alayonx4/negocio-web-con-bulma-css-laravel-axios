pues hay dos formas 


la primera es importando archivo juegos sql

es solo entrar a phpmyadmin con el local host obviamente con el xampp encendido crear tabla con el nombre juegos luego dirigirse a la 
parte de exportar y cargar el archivo juegos.sql y listo tablas creadas

la segunda es usando el setup para crearlas automaticamente

en la carpeta sistema hay que buscar el archivo setup.php hay que tener encendido el xampp para instalarlo
y tener la carpeta ventas dentro de la carpeta xampp/htdocs pues tinenes que dirigirte al buscador y escribir

localhost/ventas/sistema/setup.php

y las tablas se crearan automaticamente ahora ya podras usar el sistema o sitio web

use cualquiera de los dos metodos el que sea mejor
